## Bagaimana Cara menginstal
   * Perintah multi-baris:
      ```
         $ pkg up -y;
         $ pkg install git -y;
         $ git clone https://github.com/EAGLEBLUETEAM/EAGLE-BLUE-TEAM;
         $ cd EAGLE-BLUE-TEAM;
         $ tar -zxf assets.tar.gz;
         $ chmod +x main.sh;
      ```

   * Perintah satu baris:
      ```
         $ pkg up -y && pkg install git tar -y && git clone https://github.com/EAGLEBLUETEAM/EAGLE-BLUE-TEAM && cd EAGLE-BLUE-TEAM && tar -zxf assets.tar.gz && chmod +x main.sh;
      ```

## Bagaimana cara menjalankan
   ```
      $ bash main.sh
   ```
